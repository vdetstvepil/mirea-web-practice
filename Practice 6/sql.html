Основы SQL

Синтаксис языка запросов SQL, основные операторы

Structed Query Language

Язык запросов SQL (Structed Query Language) является языком структурированных запросов к базам данных и стандартом реляционных баз данных. Он достаточно компактен и содержит небольшое количество команд.

Самыми главными операциями языка SQL считаются следующие:

  • Создание в базе данных новых таблиц;
  • Добавление в таблицу новых записей;
  • Изменение записей;
  • Удаление записей;
  • Выборка записей из одной или нескольких таблиц;
  • Изменение структур таблиц.

    • Язык представляет собой совокупность операций, инструкций и вычисляемых функций. Согласно общепринятому стилю программирования, операторы языка SQL пишутся заглавными буквами. В конце каждого запроса принято ставить точку с запятой, которая используется в качестве разделителя при выполнении нескольких запросов подряд (очереди запросов).

    Дальнейшее рассмотрение запросов и операторов будет проводиться на примере реляционной СУБД MS SQL.

    Типы данных

    Существует несколько типов данных полей в MS SQL:

  • INTEGER - числовой тип;
  • REAL - дробный числовой тип;
  • FLOAT - дробный числовой тип;
  • DATETIME - тип даты и времени;
  • BINARY - тип двоичных данных;
  • VARCHAR - текстовый тип данных;
  • TEXT - текстовый тип данных.

    • Как правило, типы данных указываются при создании и изменении полей таблиц. В дальнейшем внесение значений в поля таблицы происходит согласно типу данных поля, и, как было рассмотрено в предыдущем разделе про типы данных, вносимые значения должны обязательно соответствовать типу данных таблицы.

    Оператор CREATE TABLE

    CREATE TABLE

    > CREATE TABLE [название таблицы] ([название столбца 1] [аргументы], [название столбца 2] [аргументы], ...);

    Допустим, нужно создать таблицу "Месяцы". В консоли прописываем соответствующие команды.

    Создание пустой таблицы "months":

    > CREATE TABLE months;

    Создание таблицы "months" со столбцами "id", "name", "days":

    > CREATE TABLE months (id INTEGER, name TEXT, days INTEGER);

    Создание таблицы "months" со столбцами "id" (ключевое поле), "name", "days":

    > CREATE TABLE months (id INTEGER PRIMARY KEY, name TEXT, days INTEGER);

    Создание таблицы "months" со столбцами "id" (ключевое поле, автоинкремент), "name", "days":

    > CREATE TABLE months (id INTEGER PRIMARY KEY IDENTITY(1,1), name TEXT, days INTEGER);

    Оператор SELECT

    SELECT

    > SELECT [название столбца 1], [название столбца 2], ... FROM [название таблицы] WHERE ... ORDER BY ... ;

    Допустим, нужно вывести значения таблицы "Месяцы". В консоли прописываем соответствующие команды.

    Вывод данных из таблицы "months":

    > SELECT * FROM months;

    Вывод значений столбца "name" из таблицы "months":

    > SELECT name FROM months;

    Вывод значений столбца "name" из таблицы "months" без повторений:

    > SELECT DISTINCT name FROM months;

    Вывод значений столбца "name" из таблицы "months", где количество дней = 30:

    > SELECT name FROM months WHERE days = 30;

    Вывод значений столбца "name" по алфавиту из таблицы "months", где количество дней = 30:

    > SELECT name FROM months WHERE days = 30 ORDER BY name;

    Оператор INSERT INTO

    INSERT INTO

    > INSERT INTO [название таблицы] ([название столбца 1], [название столбца 2], ...) VALUES ([значение 1], [значение 2], ...);

    Допустим, нужно внести данные в таблицу "months". В консоли прописываем соответствующие команды.

    Добавление записи в таблицу "months":

    > INSERT INTO months (name, days) VALUES ("February", 28);

    Добавление записи в таблицу "months", если указаны все значения в соответствии с полями:

    > INSERT INTO months VALUES (2, "February", 28);

    Оператор UPDATE

    UPDATE

    > UPDATE [название таблицы] SET [название столбца] = [значение столбца] WHERE [условие обзора];

    Допустим, нужно обновить элемент в таблице "months". В консоли прописываем соответствующие команды.

    Обновление записей в таблице "months":

    > UPDATE months SET days = 29 WHERE name = 'February';

    Оператор DELETE FROM

    DELETE FROM

    > DELETE FROM [название таблицы] WHERE [условие обзора];

    Допустим, нужно удалить элемент в таблице "months". В консоли прописываем соответствующие команды.

    Удаление записей из таблицы "months":

    > DELETE FROM months WHERE name = 'February';

    Оператор DROP TABLE

    DROP TABLE

    > DROP TABLE [название таблицы];

    Допустим, нужно удалить таблицу "months". В консоли прописываем соответствующие команды.

    Удаление таблицы "months":

    > DROP TABLE months;
